## 📝 Sobre o Projeto

O **Pontuador de Tranca** é uma ferramenta de linha de comando (CLI) projetada para substituir o papel e a caneta durante partidas de jogos de cartas. Focado em usabilidade e robustez, o sistema gerencia rodadas acumulativas e calcula automaticamente o vencedor.

Este projeto faz parte da minha jornada de transição e expansão tecnológica, aplicando conceitos de **Back-end** e lógica de programação que venho consolidando no **Bootcamp da TOTVS**.

## 🚀 Funcionalidades Técnicas

O código foi estruturado para demonstrar domínio em:

* **Validação de Entrada:** Uso de loops `while` e blocos `try/except` para impedir que dados inválidos (letras ou números fora do range) interrompam a execução.
* **Estruturas de Dados:** Implementação de **Dicionários** (`dict`) para o mapeamento eficiente de jogadores e suas respectivas pontuações.
* **Lógica de Encerramento:** Controle de fluxo dinâmico que permite ao usuário decidir o momento exato de finalizar a partida.
* **Algoritmo de Ranking:** Utilização da função `max()` com argumentos de chave para identificar o campeão no placar final.
